  18. def runWorkers[A, B](workerCount: Int)(xa: TraversableOnce[A], f: (A) ⇒ Future[B], g: (B) ⇒ Future[Unit])(implicit ec: ExecutionContext): Future[Unit]

    Using the collection members xa, concurrently execute a processing function f, outputting the result of f to the function g.

    Using the collection members xa, concurrently execute a processing function f, outputting the result of f to the function g. Concurrency is limited to the number of workers specified in WorkersConfig. Any exception that occurs during a worker's processing will cause a ConcurrentThrowable to be immediately returned. Any other exceptions from workers that have yet to complete can be retrieved from ConcurrentThrowable.

    A

    B

    the output type

    g

    the output function

    returns

    a Future that completes once all workers have completed.
