  •  package concurrent

    s_mach.concurrent is an open-source Scala library that provides asynchronous serial and parallel execution flow control primitives for working with asynchronous tasks.

    s_mach.concurrent is an open-source Scala library that provides asynchronous serial and parallel execution flow control primitives for working with asynchronous tasks. An asynchronous task consists of two or more calls to function(s) that return a future result *A ⇒ Future[B] instead of the result A ⇒ B.

    Note: only difference between 2.11 and 2.12 version is Future.unit is removed since its now part of std lib

  18.  def runWorkers[A, B](workerCount: Int)(xa: TraversableOnce[A], f: (A) ⇒ Future[B], g: (B) ⇒ Future[Unit])(implicit ec: ExecutionContext): Future[Unit]

    Using the collection members xa, concurrently execute a processing function f, outputting the result of f to the function g.

    Using the collection members xa, concurrently execute a processing function f, outputting the result of f to the function g. Concurrency is limited to the number of workers specified in WorkersConfig. Any exception that occurs during a worker's processing will cause a AsyncParThrowable to be immediately returned. Any other exceptions from workers that have yet to complete can be retrieved from AsyncParThrowable.

    returns
